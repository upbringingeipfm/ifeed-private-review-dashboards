The files below are the stable source-of-truth layers. Raw can be messy; everything after refinement must be structured.

Raw capture

/weeks/w25/raw/{tool}_output.md
Human-pasted output from each tool.

Refined register

refined-signals.json
Deduped candidate signals with date, source, claim, domain, confidence and source posture.

Dashboard data

dashboard-signals.json
Expanded cards, decision tags, filters, audit notes and private review fields.

Cockpit handoff

cockpit-handoff.json
Preliminary 20, final 9 candidates, watch, drop and reasoning.

Publish data

publish-signals.json
Only final public-safe signals, copy blocks, source URLs, visual card content and claim boundaries.
